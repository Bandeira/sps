getPairAligns - Get all the contigs pairwise alignments

  • getPairAlignsC2.m - symmetry requirement
  • getPairAlignsCe.m - experimental data
  • getPairAligns.m - non-contigs' version (OLD)
  • getPairAlignsC.m - no symmetry requirement version (OLD)

  • init.m - Starts a work session for the petide spectra alignment project
  • getScoreDistContigs.m - Generates a dataset for estimating the relevance of a score given (shift, aligned peptide's mass)
  • contigRanges.m - merge the locations of the peptides in a contig (OLD)
  • contigRanges2.m (OLD)

Data

  • AAdata.dat - Mono-isotopic amino acid masses
  • AAmassesAvg.mat - Average amino acid masses
  • AAdata_oldMasses.dat - Old table of mono-isotopic amino acid masses

  • msProfile.mat - Dancik'99 ion probabilities with integer offsets (added B2/b2-H20)
  • msProfileReal.mat - Dancik'99 ion probabilities with real offsets
  • msProfileISB.mat - ISB ion probabilities with real offsets + tolerance of 0.5 Da
  • msProfile_old.mat - Dancik'99 ion probabilities with integer offsets

  • data_spectrin.mat - spectrin fragment positions + protein sequence
  • proteinData.dat - P11217, P16278, Q9UBC5 amino acid sequences
